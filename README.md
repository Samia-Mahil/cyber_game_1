# 🛡️ مهمة الدرع السيبراني
## Cyber Shield Mission Game

لعبة تعليمية تفاعلية لطلاب **مدرسة مدينة الشامخة** حول حماية الحاسوب من الفيروسات والتهديدات الرقمية.

### 🎮 طريقة اللعب
- 5 مهام تمثّل سيناريوهات حقيقية
- اختر الإجابة الصحيحة لكسب النقاط
- كل مهمة تتضمن رابط مرجعي من مواقع الأمن السيبراني الإماراتية

### 🇦🇪 المصادر الإماراتية
- [مجلس الأمن السيبراني](https://uaecsc.gov.ae)
- [TDRA - هيئة تنظيم الاتصالات](https://tdra.gov.ae)
- [DESC - مركز دبي للأمن الإلكتروني](https://desc.gov.ae)
- [ADDA - هيئة أبوظبي للبيانات](https://adda.gov.ae)
- [aeCERT - طوارئ الحاسب](https://www.tra.gov.ae/en/about-us/aecert.aspx)

### 🚀 النشر على GitHub Pages
1. أنشئ مستودعاً جديداً
2. ارفع ملف `index.html` (غيّر اسم الملف)
3. من Settings → Pages → اختر main branch
4. الرابط: `https://username.github.io/repo-name`

### 📁 الملفات
- `cyber_game.html` → ملف اللعبة الكاملة (ملف واحد فقط، لا يحتاج اتصال إنترنت)
